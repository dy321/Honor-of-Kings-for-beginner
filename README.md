# Honor of Kings Beginner Coach

一个面向王者荣耀新手与王者 50 星以下玩家的成长型技能包。

它以“稳定成长、可执行训练、低门槛上手”为核心设计原则，优先解决最常见、最影响胜率的问题：
- 老是死
- 漏线漏经济
- 不会看小地图
- 不知道该不该支援
- 团战乱冲
- 英雄池太杂

## 适合谁

- 刚接触王者荣耀的新手
- 想从“会一点”提升到“能稳定赢”的玩家
- 王者 50 星以下，尤其是容易因为失误掉分的玩家

## 设计思路

这个 skill 的重点不是堆积复杂概念，而是帮助用户先建立稳定的实战模型：
- 少死
- 稳定发育
- 团战别先暴毙
- 先学会简单正确，再考虑复杂上限

默认回答方式强调：
1. 先指出用户当前最致命的问题
2. 给出“这局只做 3 件事”
3. 给出“这局别做 3 件事”
4. 提供一句口诀
5. 告诉用户接下来 3 把怎么练

## 核心能力

- 面向新手与中低分玩家的分层回答
- 覆盖五个分路的基础教学框架
- 支持“这局怎么打”和“怎么长期训练”两种场景
- 支持复盘型提问与结构化问题定位
- 回答更强调实战动作与训练重点，而不是抽象概念

## 使用示例

- “我玩射手总是暴毙怎么办？”
- “新手中路先练什么英雄？”
- “我老是不知道什么时候支援”
- “对抗路打不过是不是就没法玩了？”
- “给我一个适合新手的三把训练计划”
- “帮我复盘这把输在哪里”
- “我是星耀，卡在中期节奏，应该先改什么”
