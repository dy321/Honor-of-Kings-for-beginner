# 新手先改这五个坏习惯

如果你现在经常输，先别急着研究版本和复杂套路。大多数新手输，不是因为英雄太弱，而是因为坏习惯太稳定。

## 1. 线不要就去打架

最常见的问题就是兵线还在，自己先走了，结果人没帮到，线也亏了。

改法：
- 先把眼前这波线处理好
- 确定队友真能打起来，再过去
- 如果已经晚了，就不要再空跑

一句话：先吃线，后做事。

## 2. 看不到人还压得很深

很多人不是打不过，而是死得太随便。

改法：
- 小地图看不到对面中路和辅助，就后退一点
- 推线时给自己留退路
- 没位移的英雄更要保守

一句话：看不到人，就别压线。

## 3. 团战总想第一个打满伤害

新手很容易为了输出站得太靠前，结果技能还没打完就没了。

改法：
- 让前排和控制先动
- 自己先活着，再找输出位置
- 先打离自己最近的人

一句话：先活着，再输出。

## 4. 英雄池太杂

今天玩这个，明天玩那个，熟练度永远起不来。

改法：
- 一个位置先练 2 到 3 个英雄
- 同类型英雄优先，比如都玩站桩射手或都玩法师
- 输赢先看自己有没有进步，不要频繁换英雄

一句话：先玩熟，再玩花。

## 5. 一逆风就乱

逆风最怕的不是经济差，而是继续送。

改法：
- 少单走
- 优先清线
- 不乱开团
- 等队友到齐再打

一句话：逆风先守，别送更大。
